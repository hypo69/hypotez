# Модуль для создания файла SUMMARY.md

## Обзор

Модуль предназначен для автоматического создания файла `SUMMARY.md`, необходимого для компиляции документации с использованием `mdbook`. Он рекурсивно обходит указанную директорию, фильтрует файлы по языку (русский или английский) и генерирует структуру файла `SUMMARY.md` с перечислением всех найденных `.md` файлов.

## Подробней

Данный модуль автоматизирует процесс создания структуры документации для `mdbook`, что упрощает поддержку и обновление документации проекта.  Он позволяет фильтровать документацию по языкам, создавая отдельные версии документации для каждого языка.

## Функции

### `make_summary`

**Назначение**: Создает файл `SUMMARY.md`, рекурсивно обходя указанную директорию.

**Параметры**:

-   `docs_dir` (Path): Путь к исходной директории 'src'.
-   `lang` (str): Язык фильтрации файлов. Возможные значения: 'ru' или 'en'. По умолчанию 'en'.

**Возвращает**:

-   `None`: Функция ничего не возвращает.

**Вызывает исключения**:

-   `Exception`:  В случае возникновения ошибки при создании файла `summary.md`.

**Как работает функция**:

1.  Вызывает функцию `prepare_summary_path` для определения пути к файлу `SUMMARY.md`.
2.  Создает родительские директории для файла `SUMMARY.md`, если они не существуют.
3.  Вызывает функцию `_make_summary` для рекурсивного обхода директории и создания содержимого файла `SUMMARY.md`.

**Примеры**:

```python
from pathlib import Path

# Пример вызова функции make_summary
docs_dir = Path('./src')
make_summary(docs_dir, lang='ru')

# Пример вызова функции make_summary с параметром по умолчанию
docs_dir = Path('./src')
make_summary(docs_dir)
```

### `_make_summary`

**Назначение**: Рекурсивно обходит папку и создает файл `SUMMARY.md` с главами на основе `.md` файлов.

**Параметры**:

-   `src_dir` (Path): Путь к папке с исходниками `.md`.
-   `summary_file` (Path): Путь для сохранения файла `SUMMARY.md`.
-   `lang` (str): Язык фильтрации файлов. Возможные значения: 'ru' или 'en'. По умолчанию 'en'.

**Возвращает**:

-   `bool`: Возвращает `True` в случае успешного создания файла `SUMMARY.md`, `False` в случае ошибки.

**Вызывает исключения**:

-   `Exception`: В случае возникновения ошибки при записи в файл `summary.md`.

**Как работает функция**:

1.  Проверяет, существует ли файл `SUMMARY.md`, и выводит сообщение, если он будет перезаписан.
2.  Открывает файл `SUMMARY.md` для записи в кодировке UTF-8.
3.  Записывает заголовок "# Summary\n\n" в файл.
4.  Рекурсивно обходит все `.md` файлы в указанной директории.
5.  Фильтрует файлы по языку в соответствии с параметром `lang`. Файлы с суффиксом `.ru.md` обрабатываются только при `lang='ru'`, остальные - при `lang='en'`.
6.  Формирует относительный путь к каждому файлу относительно родительской директории `src_dir` и записывает его в файл `SUMMARY.md` в формате "- \[имя файла](относительный/путь/к/файлу.md)\n".

**Примеры**:

```python
from pathlib import Path

# Пример вызова функции _make_summary
src_dir = Path('./src')
summary_file = Path('./docs/SUMMARY.md')
_make_summary(src_dir, summary_file, lang='ru')

# Пример вызова функции _make_summary с параметром по умолчанию
src_dir = Path('./src')
summary_file = Path('./docs/SUMMARY.md')
_make_summary(src_dir, summary_file)
```

### `prepare_summary_path`

**Назначение**: Формирует путь к файлу `SUMMARY.md`, заменяя часть пути 'src' на 'docs' и добавляя имя файла.

**Параметры**:

-   `src_dir` (Path): Исходный путь с 'src'.
-   `file_name` (str): Имя файла, который нужно создать. По умолчанию 'SUMMARY.md'.

**Возвращает**:

-   `Path`: Новый путь к файлу.

**Как работает функция**:

1.  Формирует новый путь к файлу, заменяя часть пути `src` на `docs`.
2.  Добавляет имя файла к новому пути.

**Примеры**:

```python
from pathlib import Path

# Пример вызова функции prepare_summary_path
src_dir = Path('./src')
summary_file = prepare_summary_path(src_dir, file_name='SUMMARY.md')
print(summary_file)

# Пример вызова функции prepare_summary_path с параметром по умолчанию
src_dir = Path('./src')
summary_file = prepare_summary_path(src_dir)
print(summary_file)
