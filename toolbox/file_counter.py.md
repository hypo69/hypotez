# Модуль для подсчета строк, классов и функций в файлах

## Обзор

Модуль `file_counter.py` предназначен для рекурсивного подсчета количества строк в текстовых файлах, а также количества классов и функций, определенных в них. Он может быть использован для анализа структуры кода в директории и ее поддиректориях.

## Подробнее

Модуль предоставляет функции для подсчета строк, классов и функций в файлах, а также для определения, является ли файл бинарным. Основная функция `count_lines_in_files` рекурсивно обходит указанную директорию, подсчитывая строки, классы и функции в каждом текстовом файле. Исключаются файлы из директории `__pycache__`, Jupyter Notebook файлы (`.ipynb`) и файлы `__init__.py`.

## Функции

### `count_lines_in_files`

**Назначение**: Рекурсивно подсчитывает количество строк в текстовых файлах, а также количество классов и функций в указанной директории и ее поддиректориях.

**Параметры**:
- `directory` (str): Путь к директории.

**Возвращает**:
- `tuple[int, int, int]`: Кортеж, содержащий общее количество строк, классов и функций.

**Как работает функция**:
- Функция обходит все файлы и поддиректории в указанной директории.
- Если встречается файл, проверяется, является ли он текстовым, не находится ли он в директории `__pycache__`, не является ли он Jupyter Notebook файлом и не является ли он файлом `__init__.py`.
- Для текстовых файлов вызывается функция `count_lines_classes_functions` для подсчета строк, классов и функций.
- Если встречается директория, функция рекурсивно вызывается для этой директории.
- Результаты суммируются и возвращаются в виде кортежа.

### `is_binary`

**Назначение**: Проверяет, является ли файл бинарным.

**Параметры**:
- `filepath` (str): Путь к файлу.

**Возвращает**:
- `bool`: `True`, если файл является бинарным, `False` в противном случае.

**Как работает функция**:
- Функция пытается открыть файл в бинарном режиме (`'rb'`).
- Читает первые 512 байт файла и проверяет наличие нулевых байт (`b'\\0'`).
- Если нулевые байты обнаружены, файл считается бинарным.
- Если во время чтения файла возникает исключение, файл также считается бинарным.

**Примеры**:


### `count_lines_classes_functions`

**Назначение**: Подсчитывает количество строк, классов и функций в файле.

**Параметры**:
- `file` (file object): Файловый объект.

**Возвращает**:
- `tuple[int, int, int]`: Кортеж, содержащий количество строк, классов и функций.

**Как работает функция**:
- Функция построчно читает файл.
- Удаляет пробелы в начале и конце каждой строки.
- Если строка не пустая, увеличивает счетчик строк.
- Если строка начинается с `'class'`, увеличивает счетчик классов.
- Если строка начинается с `'def'`, увеличивает счетчик функций.
- Возвращает кортеж с количеством строк, классов и функций.
