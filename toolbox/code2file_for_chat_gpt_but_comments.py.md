# Модуль для чтения и обработки текстовых файлов
## Обзор

Модуль `code2file_for_chat_gpt_but_comments.py` предназначен для рекурсивного чтения и обработки различных типов файлов (текстовых, данных) в указанном каталоге. Он объединяет содержимое этих файлов в один выходной файл. Также предусмотрена возможность удаления файлов и директорий, имена которых начинаются с символа `_` или содержат символы `(` и `)`.

## Подробнее

Этот модуль используется для сбора и подготовки данных из различных файлов для последующей обработки, например, для анализа или обучения моделей машинного обучения. Он поддерживает различные форматы файлов, включая текстовые файлы, файлы данных (JSON, CSV, XML, Excel, YAML, SQLite) и другие.

## Функции

### `delete_directory_contents`

**Назначение**: Рекурсивно удаляет все содержимое указанной директории.

**Параметры**:
- `directory` (Path): Путь к директории, содержимое которой нужно удалить.

**Возвращает**:
- `None`

**Принцип работы**:
Функция `delete_directory_contents` принимает путь к директории и рекурсивно удаляет все файлы и поддиректории внутри этой директории. Если в процессе удаления возникают ошибки, они логируются с использованием `logger.error`.

**Примеры**:
```python
from pathlib import Path
directory_path = Path('../tmp/chat_gpt/aliexpress')
delete_directory_contents(directory_path)
```

### `read_text_files`

**Назначение**: Читает все указанные текстовые и данные файлы в каталоге и сохраняет объединенный текст в один файл, одновременно удаляя файлы и директории, которые начинаются с `_` или содержат `(` и `)`.

**Параметры**:
- `directory` (str): Каталог для поиска файлов.
- `output_file` (str): Файл, в который будет сохранен объединенный текст.

**Возвращает**:
- `None`

**Принцип работы**:
Функция `read_text_files` выполняет следующие действия:
1. Создает выходной файл, если он не существует.
2. Обходит все файлы и директории в указанном каталоге.
3. Удаляет директории и файлы, имена которых начинаются с `_` или содержат символы `(` и `)`.
4. Читает содержимое файлов различных форматов (текстовые, JSON, CSV, XML, Excel, YAML, SQLite) и записывает его в выходной файл.

**Примеры**:
```python
directory_path = 'src'
output_file_path = 'output.txt'
read_text_files(directory_path, output_file_path)
```

## Параметры

- `MODE` (str): Определяет режим работы скрипта (в данном случае 'development').
- `directory_to_search` (str): Каталог для поиска файлов. Используется в блоке `if __name__ == "__main__":`.
- `output_file_path` (Path): Файл для сохранения объединенного вывода. Используется в блоке `if __name__ == "__main__":`.

## Блок `if __name__ == "__main__":`

Этот блок кода выполняется только при запуске скрипта напрямую. Он выполняет следующие действия:

1. Указывает каталог для поиска файлов (`directory_to_search`) и файл для сохранения вывода (`output_file_path`).
2. Создает директорию для выходного файла, если она не существует.
3. Создает выходной файл, если он не существует.
4. Вызывает функцию `read_text_files` для обработки файлов в указанном каталоге и сохранения результата в выходной файл.
