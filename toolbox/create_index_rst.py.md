# Модуль `create_index_rst.py`

## Обзор

Модуль предназначен для автоматического создания файла `index.rst` в каталоге `docs` проекта. Этот файл используется для построения документации Sphinx, автоматически индексируя все Python-файлы в указанной директории и её поддиректориях.

## Подробней

Модуль рекурсивно обходит все поддиректории, начиная с указанной, находит все файлы с расширением `.py` и добавляет их в файл `index.rst` в формате `toctree`. Это позволяет автоматически генерировать содержание документации на основе структуры проекта.

## Функции

### `create_index_rst`

```python
def create_index_rst(start_dir: str) -> None:
    """ Recursively traverses all subdirectories from the start directory, reads all *.py files,
    and creates an index.rst file in the `docs` directory that lists all these files in a toctree format. Logs the process throughout.

    Args:
        start_dir (str): The root directory to start the traversal from.

    Returns:
        None

    Example:
        >>> create_index_rst(os.getcwd())
    """
    ...
```

**Назначение**: Рекурсивно обходит все поддиректории, начиная с `start_dir`, находит все файлы с расширением `.py` и создает файл `index.rst` в директории `docs`, который содержит список этих файлов в формате `toctree`.

**Параметры**:
- `start_dir` (str): Корневая директория, с которой начинается обход.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- `Exception`: В случае ошибки при создании или записи в файл `index.rst`.

**Как работает функция**:

1.  Определяет начальный путь и путь к директории `docs`, а также путь к файлу `index.rst`.
2.  Проверяет существование директории `docs` и создает её, если она отсутствует.
3.  Открывает файл `index.rst` для записи.
4.  Записывает в файл заголовки `toctree` для Sphinx.
5.  Рекурсивно обходит все поддиректории, начиная с `start_dir`.
6.  Находит все файлы с расширением `.py` в каждой директории. Исключает файлы, содержащие символы `(` и `)` в имени.
7.  Для каждого найденного файла формирует относительный путь и записывает его в файл `index.rst`.
8.  Логирует процесс создания файла и добавления файлов в него.
9.  Обрабатывает исключения, которые могут возникнуть в процессе создания или записи в файл.

**Примеры**:

```python
import os
from pathlib import Path

# Пример использования функции
current_dir = os.getcwd()  # Получаем текущую рабочую директорию
create_index_rst(current_dir)
```

## Пример использования

В конце модуля находится пример использования функции `create_index_rst`:

```python
if __name__ == "__main__":
    create_index_rst(Path(header.__root__, \'src\'))
```

Этот код запускает функцию `create_index_rst` с путем, полученным из переменной `__root__` модуля `header` и поддиректории `src`.
