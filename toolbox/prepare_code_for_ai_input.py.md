# Модуль для подготовки кода к вводу в AI-модели
## Обзор

Модуль содержит функции для сбора содержимого файлов из указанной директории, исключая нежелательные директории и файлы, и сохранения их в один текстовый файл, предназначенный для ввода в модели машинного обучения. Поддерживаются файлы с расширениями `.py`, `.json`, `.md`, `.dot` и `.mer`.

## Подробнее

Этот модуль предназначен для автоматизации процесса подготовки данных для обучения AI-моделей. Он рекурсивно сканирует указанную директорию, собирает содержимое файлов с определенными расширениями и сохраняет их в единый текстовый файл. Это упрощает задачу предоставления AI-моделям необходимой информации о структуре проекта и коде.

## Функции

### `collect_file_contents`

**Назначение**: Рекурсивно собирает содержимое файлов из указанной директории.

**Параметры**:
- `directory` (Path): Путь к исходной директории для сканирования.
- `target_directory` (Path): Путь к директории, в которой будет сохранен итоговый текстовый файл для ввода в ML-модель.

**Возвращает**:
- `dict`: Словарь, где ключами являются пути к файлам, а значениями - содержимое файлов.

**Как работает функция**:
- Функция `collect_file_contents` рекурсивно обходит указанную директорию, используя `directory.iterdir()`.
- Для каждого элемента (файла или директории) проверяется, является ли он директорией.
- Если элемент является директорией, и его имя не входит в список исключений (`profiles`, `__pycache__`, `_experiments`) и не начинается с `___`, и не содержит `*`, функция рекурсивно вызывает саму себя для этой директории, обновляя словарь `contents`.
- Если элемент является файлом, проверяется его расширение. Если расширение файла входит в список разрешенных (`.py`, `.json`, `.md`, `.dot`, `.mer`) и имя файла не начинается с `___`, и не содержит `*` и `(` и `)`, файл открывается для чтения с кодировкой `utf-8`, его содержимое считывается и добавляется в словарь `contents`, где ключом является путь к файлу.
- В конце функция возвращает словарь `contents`, содержащий пути к файлам и их содержимое.

**Примеры**:
```python
from pathlib import Path
# Пример вызова функции collect_file_contents
directory = Path('./src')
target_directory = Path('./output')
contents = collect_file_contents(directory, target_directory)
print(contents)
```

### `save_contents_to_text`

**Назначение**: Сохраняет собранное содержимое файлов в один текстовый файл.

**Параметры**:
- `contents` (dict): Словарь с путями к файлам в качестве ключей и содержимым файлов в качестве значений.
- `output_file` (Path): Путь к выходному текстовому файлу для ввода в ML-модель.

**Как работает функция**:
- Функция `save_contents_to_text` открывает указанный `output_file` для записи с кодировкой `utf-8`.
- Она итерируется по словарю `contents`, полученному от функции `collect_file_contents`.
- Для каждого файла она записывает в файл строку "File: {path}\n", где `path` - путь к файлу.
- Затем она записывает содержимое файла.
- После содержимого каждого файла добавляется разделитель в виде строки из 80 символов `=` и двух переносов строки.

**Примеры**:
```python
from pathlib import Path
# Пример вызова функции save_contents_to_text
contents = {'./file1.txt': 'content1', './file2.txt': 'content2'}
output_file = Path('./output.txt')
save_contents_to_text(contents, output_file)
```

### `main`

**Назначение**: Главная функция для запуска сбора содержимого и сохранения в текстовый файл.

**Как работает функция**:
- Функция `main` определяет путь к исходной директории (`src_directory`) как `Path(header.__root__, 'src', 'utils')`.
- Создает путь к директории для сохранения результатов (`project_structure_directory`) как `Path(src_directory, 'prod')`.
- Создает директорию `project_structure_directory`, если она не существует, используя `project_structure_directory.mkdir(parents=True, exist_ok=True)`.
- Вызывает функцию `collect_file_contents` для сбора содержимого файлов из `src_directory` и сохраняет результат в переменной `file_contents`.
- Определяет путь к выходному файлу (`output_file_path`) как `Path(project_structure_directory, 'all_file_contents.txt')`.
- Вызывает функцию `save_contents_to_text` для сохранения собранного содержимого в файл `output_file_path`.

**Примеры**:
```python
# Пример вызова функции main
main()
```
