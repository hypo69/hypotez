# Модуль ai: Интерфейс управления моделями ИИ
============================================

Модуль `ai` представляет собой интерфейс для управления различными моделями искусственного интеллекта. Он абстрагирует взаимодействие с внешними API и предоставляет унифицированный способ настройки и использования моделей для анализа данных и обработки естественного языка.

Модуль включает следующие подмодули:

1. **anthropic**
   Интеграция с моделями Anthropic для продвинутого понимания языка и генерации ответов.
   [Перейти к модулю](./anthropic/readme.ru.md)

2. **dialogflow**
   Интеграция с Google Dialogflow для обработки естественного языка (NLU) и создания интерактивных разговорных приложений.
   [Перейти к модулю](./dialogflow/readme.ru.md)

3. **gemini**
   Управление соединениями с моделями Gemini, обеспечивая поддержку для приложений, требующих уникальных возможностей ИИ Gemini.
   [Перейти к модулю](./gemini/readme.ru.md)

4. **helicone**
   Подключение к моделям Helicone, предоставляя доступ к специализированным функциям для настройки решений на базе ИИ.
   [Перейти к модулю](./helicone/readme.ru.md)

5. **llama**
   Интерфейс для LLaMA (Large Language Model Meta AI), предназначен для задач, связанных с пониманием и генерацией естественного языка в различных приложениях.
   [Перейти к модулю](./llama/readme.ru.md)

6. **myai**
   Кастомный подмодуль ИИ, разработанный для специализированных конфигураций моделей и реализации, обеспечивающий уникальные функции ИИ, специфичные для проекта.
   [Перейти к модулю](./myai/readme.ru.md)

7. **openai**
   Интеграция с API OpenAI, предоставляя доступ к их набору моделей (например, GPT) для задач генерации текста, классификации, перевода и других.
   [Перейти к модулю](./openai/readme.ru.md)

8. **tiny_troupe**
   Интеграция с моделями ИИ от Microsoft, предлагая решения для обработки естественного языка и задач анализа данных с использованием маленьких моделей, оптимизированных для производительности.
   [Перейти к модулю](./tiny_troupe/readme.ru.md)

9. **revai**
    Интеграция с моделью от rev.com, специализирующейся на работе с аудиофайлами, такими как записи переговоров, совещаний, звонков и других аудио-материалов.
    [Перейти к модулю](./revai/readme.ru.md)

10. **prompts**
    Системные и командные промпты в формате `markdown` для моделей ИИ.

**Пример использования:**

```python
# Пример использования подмодуля openai для генерации текста
from src.ai import openai

openai.generate_text("Напиши короткое стихотворение о весне.")
```
