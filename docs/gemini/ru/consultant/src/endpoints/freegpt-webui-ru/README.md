   ## Описание

   FreeGPT WebUI предоставляет доступ к ChatGPT с вашего ПК без использования VPN. В настоящее время поддерживаются:

   - ChatGPT 3.5 Turbo
   - Веб-интерфейс для взаимодействия с ботом
   - Набор джейлбрейков для расширения возможностей

   ## Установка

   ### Windows

   1.  **Скачайте и установите:**
       -   [git](https://git-scm.com/download/win)
       -   [Python 3.10.X](https://www.python.org/downloads/). Убедитесь, что Python добавлен в PATH.
       -   При возникновении ошибок, связанных с Visual Studio, скачайте [Visual Studio](https://visualstudio.microsoft.com/ru/downloads/) для компиляции библиотек.

   2.  **Клонируйте репозиторий:**

       Откройте командную строку (терминал) в папке, где хотите разместить `freegpt-webui-ru`, и выполните команду:

       ```
       git clone https://github.com/Em1tSan/freegpt-webui-ru.git
       ```

   3.  **Запустите установку:**

       Откройте папку `freegpt-webui-ru` и запустите файлы `install.bat`, а затем `start.bat`. Скрипт создаст виртуальное окружение, установит зависимости и запустит приложение.

   4.  **Откройте в браузере:**

       Перейдите по адресу: http://127.0.0.1:1338

   ### Портативная версия

   Портативная версия не требует установки Python. Скачать её можно в [релизах](https://github.com/Em1tSan/freegpt-webui-ru/releases). Просто распакуйте архив и запустите приложение.
