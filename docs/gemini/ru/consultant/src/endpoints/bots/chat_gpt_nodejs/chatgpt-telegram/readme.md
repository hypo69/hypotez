### Голосовой помощник ChatGPT - Telegram

Этот проект представляет собой Telegram-бота, использующего ChatGPT для обработки голосовых сообщений.

#### Описание

Бот позволяет пользователям отправлять голосовые сообщения, которые затем транскрибируются и отправляются в ChatGPT для получения ответа. Ответ ChatGPT отправляется обратно пользователю в текстовом формате.

#### Ссылка на статью

[Статья на Хабре](https://habr.com/ru/companies/selectel/articles/731692/) содержит подробное описание процесса создания и настройки бота.

#### Инструкции по установке

1.  **Клонируйте репозиторий:**

    ```bash
    git clone <repository_url>
    cd <repository_directory>
    ```
2.  **Установите зависимости:**

    ```bash
    pip install -r requirements.txt
    ```
3.  **Настройте переменные окружения:**

    -   `TELEGRAM_BOT_TOKEN`: Токен вашего Telegram-бота.
    -   `OPENAI_API_KEY`: API-ключ OpenAI.
    -   `OPENAI_MODEL`: Используемая модель OpenAI (например, `gpt-3.5-turbo`).

4.  **Запустите бота:**

    ```bash
    python bot.py
    ```

#### Использование

1.  Запустите Telegram-бота.
2.  Отправьте боту голосовое сообщение.
3.  Бот автоматически транскрибирует сообщение и отправит его в ChatGPT.
4.  Бот отправит вам ответ от ChatGPT в текстовом формате.

#### Дополнительная информация
