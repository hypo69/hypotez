```markdown
### **Инструкции для бота**

Описание команд, доступных для взаимодействия с ботом.

**Команды**:

*   `!hi`: Приветствует пользователя.
*   `!train <данные> <директория_данных> <метка_позитивного_класса> <вложение>`: Обучает модель на предоставленных данных. Используйте `<данные>` для файла, `<директория_данных>` для директории с данными, или `<вложение>` для прикрепленного файла.
    *   `<данные>`: Путь к файлу с данными для обучения (например, `data.txt`).
    *   `<директория_данных>`: Путь к директории, содержащей файлы с данными для обучения (например, `data_dir/`).
    *   `<метка_позитивного_класса>`: Значение, обозначающее позитивный класс (например, `positive`).
    *   `<вложение>`: Файл, прикрепленный к сообщению, который содержит данные для обучения.
    *   **Пример**: `!train data.txt data_dir positive attachment`
*   `!test <тестовые_данные>`: Тестирует модель с предоставленными JSON тестовыми данными.
    *   `<тестовые_данные>`: JSON строка, содержащая данные для тестирования модели.
    *   **Пример**: `!test {"feature1": 1, "feature2": 2}`
*   `!archive <директория>`: Архивирует файлы в указанной директории.
    *   `<директория>`: Путь к директории, которую необходимо заархивировать (например, `archive_dir/`).
    *   **Пример**: `!archive archive_dir/`
*   `!select_dataset <путь_к_директории_позитивных_данных> <метка_позитивного_класса>`: Выбирает датасет для обучения из указанной директории.
    *   `<путь_к_директории_позитивных_данных>`: Путь к директории, содержащей позитивные данные (например, `positive_data/`).
    *   `<метка_позитивного_класса>`: Значение, обозначающее позитивный класс (например, `positive`).
    *   **Пример**: `!select_dataset positive_data/ positive`
*   `!instruction`: Отображает это сообщение с инструкциями.

**Возможные ошибки и их обработка**:

*   Если указан неверный путь к файлу или директории, бот выдаст сообщение об ошибке.
*   Если данные для обучения или тестирования имеют неверный формат, бот выдаст сообщение об ошибке.
*   В случае возникновения других ошибок, бот уведомит пользователя о произошедшем.
