# Модуль для реализации чат-бота ChatGPT на Node.js

## Описание

Этот модуль содержит информацию и инструкции по реализации чат-бота ChatGPT с использованием Node.js. Он основан на статье [Реализация чат-бота на Node.js](https://habr.com/ru/companies/selectel/articles/731692/).

## Функциональность

Модуль предоставляет:

- Пример кода для создания простого чат-бота.
- Инструкции по настройке и запуску бота.
- Описание основных компонентов и принципов работы.

## Пример использования

1. **Установка зависимостей:**

   ```bash
   npm install openai
   ```

2. **Настройка API ключа:**

   ```javascript
   const OpenAI = require('openai');

   const openai = new OpenAI({
     apiKey: 'YOUR_API_KEY', // Замените на свой API ключ
   });
   ```

3. **Создание запроса к ChatGPT:**

   ```javascript
   async function main() {
     const completion = await openai.chat.completions.create({
       messages: [{ role: "system", content: "You are a helpful assistant." }, { role: "user", content: "Hello!" }],
       model: "gpt-3.5-turbo",
     });

     console.log(completion.choices[0].message.content);
   }

   main();
   ```
