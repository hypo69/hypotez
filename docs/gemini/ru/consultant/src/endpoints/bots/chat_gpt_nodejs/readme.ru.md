## Описание

Этот модуль содержит код и инструкции для создания и запуска чат-бота ChatGPT на платформе Node.js.
Он использует библиотеки и API Node.js для взаимодействия с моделью ChatGPT и обработки входящих сообщений.

## Функциональность

-   Обработка входящих текстовых сообщений
-   Взаимодействие с API ChatGPT
-   Генерация ответов на основе модели ChatGPT
-   Интеграция с другими приложениями и сервисами

## Зависимости

Для работы модуля требуются следующие библиотеки и модули Node.js:

-   `axios`: для выполнения HTTP-запросов к API ChatGPT
-   `body-parser`: для обработки входящих запросов
-   `express`: для создания веб-сервера

## Использование

### Пример кода

```javascript
const express = require('express');
const bodyParser = require('body-parser');
const axios = require('axios');

const app = express();
app.use(bodyParser.json());

app.post('/chat', async (req, res) => {
    const message = req.body.message;

    try {
        const response = await axios.post('https://api.openai.com/v1/chat/completions', {
            model: 'gpt-3.5-turbo',
            messages: [{ role: 'user', content: message }],
        }, {
            headers: {
                'Authorization': 'Bearer YOUR_API_KEY',  // Замените на свой API-ключ
                'Content-Type': 'application/json',
            },
        });

        const answer = response.data.choices[0].message.content;
        res.send({ answer: answer });
    } catch (error) {
        console.error(error);
        res.status(500).send({ error: 'Произошла ошибка при обработке запроса.' });
    }
});

const PORT = 3000;
app.listen(PORT, () => {
    console.log(`Сервер запущен на порту ${PORT}`);
});
```

### Описание API

#### `POST /chat`

Обрабатывает входящие сообщения и возвращает ответ от ChatGPT.

**Параметры запроса:**

-   `message` (string): Текст сообщения для ChatGPT.

**Возвращаемый ответ:**

-   `answer` (string): Ответ от ChatGPT.
