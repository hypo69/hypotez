# Toolbox AI

## Описание

Toolbox AI - это универсальный ассистент, который может генерировать контент для различных рабочих задач.

## Возможности

- Создание уникальных текстов для SMM, email-рассылок, SEO-продвижения, рекламных кампаний и многого другого.
- Генерация креативных идей для реализации.
- Создание визуального контента для постов, баннеров и иллюстраций.
- Автоматическая транскрипция подкастов, вебинаров и видео.

## Инструкция по использованию

1. Выберите нужную команду.
2. Введите необходимые входные данные.
3. Получите готовый контент.

## Начало работы

Для ознакомления с сервисом предоставляется 5 бесплатных генераций. После этого можно выбрать тарифный план, который покрывает все ваши рабочие задачи.

## Техническая информация

### Архитектура

Toolbox AI состоит из следующих основных компонентов:

- **Модуль обработки текста**: отвечает за генерацию и обработку текстового контента.
- **Модуль генерации идей**: предлагает креативные концепции для реализации.
- **Модуль генерации изображений**: создает визуальный контент на основе текстового описания.
- **Модуль транскрипции**: автоматическая расшифровка аудио- и видеофайлов.

### Используемые технологии

- Python 3.9+
- [TensorFlow](https://www.tensorflow.org/) / [PyTorch](https://pytorch.org/) (для моделей машинного обучения)
- [Transformers](https://huggingface.co/transformers/) (для работы с нейронными сетями)
- [FFmpeg](https://ffmpeg.org/) (для транскрипции аудио и видео)
- src.logger (для логирования)

### Установка

1. Клонируйте репозиторий:
   ```bash
   git clone <repository_url>
   ```
2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/macOS
   venv\Scripts\activate  # Windows
   ```
3. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
   ```

### Настройка

1. Создайте файл конфигурации `config.json` на основе примера `config.example.json`.
2. Укажите необходимые параметры, такие как ключи API для используемых сервисов (например, OpenAI, Google Gemini).

### Запуск

```bash
python main.py
```

### Структура проекта

```
toolbox_ai/
├── main.py             # Основной файл запуска
├── modules/            # Модули, реализующие основную функциональность
│   ├── text_processing/  # Модуль обработки текста
│   │   ├── generator.py    # Генерация текста
│   │   └── ...
│   ├── idea_generation/  # Модуль генерации идей
│   │   └── ...
│   ├── image_generation/ # Модуль генерации изображений
│   │   └── ...
│   ├── transcription/    # Модуль транскрипции
│   │   └── ...
│   └── ...
├── config.json         # Файл конфигурации
├── requirements.txt    # Список зависимостей
├── README.md           # Этот файл
└── ...
```

### Добавление новой функциональности

1. Создайте новый модуль в директории `modules/`.
2. Реализуйте необходимую функциональность.
3. Добавьте новый модуль в основной файл `main.py`.
4. Настройте новый модуль в файле конфигурации `config.json`.

### Тестирование

1. Напишите тесты для нового модуля.
2. Запустите тесты:
   ```bash

   ```
