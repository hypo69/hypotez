# Movie Bot

Телеграм бот для поиска ссылок на бесплатный просмотр интересующих фильмов и сериалов.
Бот работает на библиотеке Aiogram, использует сервисы google для ассоциативного поиска,
а также библиотеку Кинопоиска и сервис [w2.kpfr/wiki](https://w2.kpfr.wiki/).
В боте реализована защита от флудинга с помощью middlewares. Для парсинга используется BeautifulSoup.

---

### Описание

Этот Telegram-бот предназначен для поиска ссылок на бесплатный просмотр фильмов и сериалов. Он использует библиотеку `Aiogram` для управления ботом, сервисы Google для поиска, библиотеку `Кинопоиска` и сервис `w2.kpfr/wiki` для получения информации о фильмах.

### Структура проекта

-   `run.py`: Основной файл для запуска бота.
-   `src/`: Содержит исходный код бота.
    -   `handlers/`: Обработчики команд и событий.
    -   `middlewares/`: Промежуточное ПО для обработки запросов (например, защита от флуда).
    -   `logger.py`: Модуль для логирования событий и ошибок.
-   `requirements.txt`: Список зависимостей проекта.
-   `.env`: Файл с переменными окружения (например, токен бота).

### Зависимости

-   `Aiogram`: Для управления Telegram-ботом.
-   `BeautifulSoup`: Для парсинга HTML-страниц.
-   `Кинопоиск API`: Для получения информации о фильмах.
-   `w2.kpfr/wiki`: Сервис для поиска ссылок на фильмы.

### Быстрый старт

1.  Склонируйте проект на свой компьютер:

    ```shell
    git clone <repository_url>
    ```
2.  Создайте файл `.env` и наполните его в предложенном формате:

    ```
    TOKEN=123456789  # Токен вашего telegram бота
    ```
3.  Если у вас нет виртуального окружения, создайте и активируйте его:

    ```shell
    python -m venv venv
    ```

    ```shell
    venv\Scripts\activate.bat  # Для Windows
    # или
    source venv/bin/activate   # Для Linux/macOS
    ```
4.  Обновите pip и установите необходимые зависимости:

    ```shell
    pip install --upgrade pip
    ```

    ```shell
    pip install -r requirements.txt
    ```
5.  Запустите `run.py`:

    ```shell
    python run.py
    ```

### Обработка ошибок и логирование

В проекте используется модуль `logger` из `src.logger` для логирования событий и ошибок. Ошибки логируются с использованием `logger.error`.

Пример:

```python
from src.logger import logger

try:
    # Код, который может вызвать ошибку
    ...
except Exception as ex:
    logger.error('Произошла ошибка при обработке данных', ex, exc_info=True)
```

### Защита от флуда

В боте реализована защита от флуда с помощью middlewares. Это позволяет предотвратить злоупотребление ботом и обеспечить его стабильную работу.
