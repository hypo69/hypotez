## Как сообщать об ошибках и получать помощь

В этом проекте GitHub Issues используется для отслеживания ошибок и запросов на новые функции. Пожалуйста, выполните поиск по существующим проблемам, прежде чем сообщать о новых, чтобы избежать дублирования. Для новых проблем, пожалуйста, зарегистрируйте свою ошибку или запрос функции в качестве новой Issue.

Для получения помощи и ответов на вопросы об использовании этого проекта, пожалуйста, обратитесь к нашей Wiki или опубликуйте сообщение на нашем форуме обсуждений, оба находятся здесь, на GitHub.
