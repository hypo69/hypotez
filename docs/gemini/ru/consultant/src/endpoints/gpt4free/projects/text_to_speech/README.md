# Модуль text_to_speech
=================================================

Модуль предназначен для преобразования текста в речь с использованием библиотеки transformers.js.

Пример использования
----------------------

Для использования модуля необходимо установить зависимости и выполнить преобразование текста в речь.

```bash
npm install @xenova/transformers
```

```python
from transformers import pipeline

# Инициализация pipeline для text-to-speech
synthesizer = pipeline("text-to-speech", "facebook/fastspeech2-en-ljspeech")

# Преобразование текста в речь
text = "Hello, this is a text-to-speech demo."
speech = synthesizer(text)

# Сохранение результата в файл
with open("output.wav", "wb") as f:
    f.write(speech["audio"])
```
