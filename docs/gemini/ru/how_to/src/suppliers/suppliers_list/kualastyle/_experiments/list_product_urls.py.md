### Как использовать этот блок кода
=========================================================================================

Описание
-------------------------
Данный блок кода содержит список URL-адресов товаров магазина Kualastyle. Эти URL-адреса используются для сбора информации о товарах, такой как название, цена, описание и изображения.

Шаги выполнения
-------------------------
1. Определяется список `product_urls`, который содержит строковые значения URL-адресов товаров.
2. Каждый URL в списке указывает на страницу конкретного товара на сайте Kualastyle.
3. Этот список может быть использован в дальнейшем для итерации по URL-адресам и извлечения данных о товарах с каждой страницы.
